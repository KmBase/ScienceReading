
# 开发

**注意**：核心代码已加密，暂不提供

## 创建并激活venv

```bash
cd SciReadingDownloader
python -m venv venv
venv\Scripts\activate.bat
```

## 安装依赖

```bash
pip install -r requirements.txt
```

## 执行脚本

```bash
python main.py
```

## 打包脚本

```bash
python build.py
```
